## SQL Analytics Overview

This folder contains SQL Server scripts used to transform raw sales and marketing data into business-ready analytical tables.

### Key Analyses

- Profit analysis (time, region, channel)
- Marketing performance (CTR, CPC, ROI, ROAS)
- Campaign seasonal effectiveness
- Customer cohort retention & CLV

### Techniques Used

- CTEs
- Window Functions (LAG, RANK, DENSE_RANK)
- Cohort analysis
- Pareto analysis
- Campaign scoring models
